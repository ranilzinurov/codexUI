    Справка

    Откройте нужную страницу, нажмите Inject Overlay и выберите элементы прямо на странице.

    Комментарии и голосовые заметки вводятся inline в overlay. В очереди остаются только выбранные элементы, порядок и отправка.

    DevTools включайте только когда нужны console/network данные. Checkbox Bodies добавляет тела запросов и ответов, поэтому используйте его только для безопасных страниц.

    Server URL и Pairing token находятся во вкладке Settings.
