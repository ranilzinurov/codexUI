    Справка

    Откройте нужную страницу, нажмите Pick on Page и выберите элементы прямо на странице.

    Комментарии и голосовые заметки вводятся inline в overlay. В очереди остаются только выбранные элементы, порядок и отправка.

    Diagnostics включайте только когда нужны console/network данные. Checkbox Bodies добавляет тела запросов и ответов, поэтому используйте его только для безопасных страниц.

    Server URL и Browser binding code находятся во вкладке Settings.
