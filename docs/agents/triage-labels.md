# Triage Labels

Use these GitHub labels for the canonical triage states:

- `needs-triage` — maintainer needs to evaluate
- `needs-info` — waiting on reporter
- `ready-for-agent` — fully specified and ready for an AFK agent
- `ready-for-human` — needs human interaction or acceptance
- `wontfix` — will not be actioned

When labels are missing, prefer creating the canonical label instead of inventing a near-duplicate.
