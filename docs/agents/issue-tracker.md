# Issue Tracker

Issues for this repository are tracked in GitHub Issues.

- Repository: `ranilzinurov/codexUI`
- Create issues with `gh issue create --repo ranilzinurov/codexUI`
- Read issues with `gh issue view --repo ranilzinurov/codexUI <number>`
- List issues with `gh issue list --repo ranilzinurov/codexUI`

Use GitHub issue numbers and URLs as durable references in plans, PRs, and follow-up work.
